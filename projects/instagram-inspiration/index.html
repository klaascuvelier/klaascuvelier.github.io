Instagram Inspiration

chrome-inspiration is a Chrome extension displaying inspirational images on your new tabs. Being inspirated is a great way to stay motivated to achieve goals.

Using Instagram's hashtags you'll be able to display photos and videos from the categories you find most interesting. It's also possible to just display your feed or media suggested by Instagram.

Developer

Instagram Inspiration (chrome-inspiration) is created by Klaas Cuvelier as a fun side project.

The only goal of this browser extension is to create a nice experience for the end user by providing inspirational images and videos using Instagram's media.
